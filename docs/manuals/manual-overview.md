# TrustBridge Kenya Manual Overview

TrustBridge Kenya is a role-based freelance trust marketplace for Kenyan freelancers, clients, members, and admins.

## Main roles

| Role | Main purpose | Main workspace |
|---|---|---|
| Guest | Browse jobs and talent before joining | Public pages |
| Member | Start with beginner access | Member dashboard |
| Freelancer | Build a profile, apply for jobs, submit verification | Freelancer dashboard |
| Client | Post jobs, search talent, invite freelancers | Client dashboard |
| Admin | Control users, jobs, applications, verification, trust score, and audit | Admin dashboard |
