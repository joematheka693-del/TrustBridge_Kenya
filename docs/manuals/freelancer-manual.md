# Freelancer Manual

Freelancers use TrustBridge Kenya to build profiles, apply for jobs, submit verification evidence, and grow trust scores.

## 1. Freelancer access

Login using a freelancer account. Demo account:

```txt
Email: freelancer@trustbridge.co.ke
Password: Freelancer@12345
```

After login, the app opens the freelancer dashboard.

## 2. Freelancer dashboard

The freelancer dashboard should show:

| Section | Purpose |
|---|---|
| Profile builder | Create or improve talent profile |
| Available jobs | Find work opportunities |
| Applications | Track job applications |
| Verification | Submit evidence |
| Trust score | Review reputation progress |
| Growth target | Guide next improvement steps |

## 3. Building a profile

Go to `/profile-builder`.

A strong profile should include:

| Field | Recommendation |
|---|---|
| Name | Use your professional name |
| Category | Choose your main work category |
| Skill level | Beginner, intermediate, advanced, or expert |
| Location | City/country or remote |
| Rate | Hourly or project rate |
| Bio | Short clear summary of your services |
| Skills | Add specific skills |
| Portfolio links | Add GitHub, website, Behance, certificates, or proof links |

Freelancers can create and update their own profile. Clients cannot create profiles.

## 4. Applying for jobs

Go to `/jobs`, open a job, and start a job application.

Application fields include:

| Field | Meaning |
|---|---|
| Message | Explain why you fit the job |
| Budget | Your price or proposal amount |
| Timeline | How long you need |

Applications start as `pending`.

## 5. Verification

Go to `/verification` or `/verification/new`.

Recommended evidence:

| Evidence type | Good example |
|---|---|
| Portfolio link | Live project, GitHub repo, Behance portfolio |
| GitHub link | Active code profile |
| Certificate link | Training certificate |
| Previous work proof | Screenshot or public project link |

## 6. Trust score

Trust score grows when you complete positive actions.

Examples:

| Action | Points |
|---|---:|
| Approved verification | +15 |
| Talent profile completed | +8 |
| Job application submitted | +2 |
| Rejected verification | -10 |

## 7. Freelancer restrictions

Freelancers cannot post jobs. Freelancers cannot moderate users or access admin pages. Freelancers can only manage their own profile and application records.
